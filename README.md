# Neon Tetris 🎮

A modern, mobile-compatible Tetris-inspired game with neon aesthetics.

**Live:** https://neon-tetris.pages.dev (connect via Cloudflare Pages)

## Features
- Dark neon theme with glow effects
- Touch controls for mobile + keyboard for desktop
- Score tracking, level system, increasing speed
- Next piece preview, ghost piece
- Particle effects on line clears
- Web Audio API sound effects
- Pause/resume, game over with restart

## Play
Open `index.html` in any browser. Works offline.
